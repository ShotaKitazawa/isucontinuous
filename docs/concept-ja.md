# Concept

isu-continuous は複数台のサーバを用いた複数人での開発を支援するツールです。
isu-continuous は複数台のサーバ上にあるソースコードを取り込み、継続的なデプロイメント及びプロファイリングを実現します。

isu-continuous には以下のそれぞれのフェーズを意識したコマンド体系が用意されています。

* `init`: Git ローカルリポジトリを初期化する
